This application will let you create, edit, and delete Categories and Items within those Catagories all inside of a catalog. 

To Use:
1. Clone this repository to your local machine.
2. Run the command "python application.py" in the command line while in the "catalog" directory.
3. Open your browser and go to "http://localhost:5000"

From there, the UI should be intuitive to create Categories and Items in your catalog.
